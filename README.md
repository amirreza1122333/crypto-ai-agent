# 🚀 Crypto AI Agent

AI-powered system for analyzing and ranking cryptocurrency markets using real-time data.

## 🔧 Tech Stack
- Python
- FastAPI
- Scikit-learn
- Pandas / NumPy
- Telegram Bot API

## ⚙️ Features
- Real-time crypto data from CoinGecko
- AI-based scoring system
- Volume/Market Cap analysis
- Risk classification
- Telegram alerts

## ▶️ How to Run

```bash
pip install -r requirements.txt
uvicorn app.api:app --reload
